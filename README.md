# 全国企业信用信息公示系统（广西）增强
（目前仅针对个体工商户年报页面开发）

安装方式：
Google Chrome/360安全浏览器/360极速浏览器/Edge等：
- **旧浏览器使用chrome-mv3.crx；新浏览器选择chrome-mv3.crx**
- 浏览器-扩展程序-管理扩展程序（或在地址栏中输入 chrome://extensions）-启用开发者模式
- 下载chrome-mvX.crx-浏览器-扩展程序-将crx拖到浏览器-添加扩展程序
- 教程1[Chrome浏览器\Edge浏览器\安装扩展程序教程](https://cloud.tencent.com/developer/article/2529970)
- 教程2[什么？都2024年了你还不知道浏览器插件怎么安装？新手必看的扩展插件安装指南](https://blog.csdn.net/qq_46438511/article/details/135724172)

浏览器已安装篡改猴 (Tampermonkey)/油猴(Greasemonkey)等插件：
- 下载gxqyxygs-enhance.user.js-将js拖到浏览器-安装

卸载方式：
Google Chrome/360安全浏览器/360极速浏览器/Edge等：
- 浏览器-扩展程序-管理扩展程序-卸载/移除

篡改猴 (Tampermonkey)/油猴(Greasemonkey)等插件：
- 已安装脚本-删除

## 功能

- 填写页：
  - 自动填写由【注册页】或【变更页】保存的【（新）联络员手机号码】
  - 于顶部添加一键预览并公示按钮免于滚动到底部
  - 于顶部添加填写默认按钮可填充默认数值（0）和选项（否）

- 变更页：
  - 自动填写由【登录页】保存的【统一社会信用代码/注册号】【原联络员证件号码】【原联络员姓名】和【原联络员手机号码】
  - 添加【联络员信息】与【法定代表人（负责人）信息】同步按钮
  - 自动保存【统一社会信用代码/注册号】和【联络员证件号码】用于跳转【登录页】时自动填写
  - 自动保存【新联络员手机号码】用于跳转【填写页】时自动填写

- 登录页：
  - 自动填写由【注册页】保存的【统一社会信用代码/注册号】和【联络员证件号码】
  - 添加【统一社会信用代码/注册号】开头选择框并自动拼接选择框和输入框内容
  - 添加【联络员证件号码】开头选择框并自动拼接选择框和输入框内容
  - 自动保存【统一社会信用代码/注册号】和【联络员证件号码】用于跳转【注册页】时自动填写
  - 自动保存【统一社会信用代码/注册号】、【联络员证件号码】、【企业联络员姓名】和【企业联络员手机号】用于跳转【变更页】时自动填写

- 注册页：
  - 自动填写由【登录页】保存的【统一社会信用代码/注册号】和【联络员证件号码】
  - 添加【联络员证件号码】与【法定代表人（负责人）证件号码】、【联络员姓名】与【法定代表人（负责人）姓名】互相同步按钮
  - 自动保存【统一社会信用代码/注册号】和【联络员证件号码】用于跳转【登录页】时自动填写
  - 自动保存【联络员手机号码】用于跳转【填写页】时自动填写

- 选项页：
  - 于顶部添加一键进入填写按钮免于勾选和选择年份

- 预览页：
  - 于顶部添加一键提交并公示按钮免于滚动到底部
